# Simple Vue.js Project

A minimal Vue.js application to get you started quickly.

## Features

- Simple todo/list app built with Vue 3
- Add and remove items from a list
- Real-time item counter
- Clean, responsive UI

## How to Use

1. Open `index.html` in your web browser
2. Type an item name and click "Add Item" (or press Enter)
3. View your items in the list below
4. Click "Delete" to remove any item

## Project Structure

```
.
└── index.html          # Main Vue.js application
```

## Technologies

- **Vue.js 3** - Progressive JavaScript framework
- **HTML5** - Markup structure
- **CSS3** - Styling and layout

## Running the Project

Simply open the `index.html` file in any modern web browser. No build tools or server required!

Alternatively, you can use a simple HTTP server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Notes

- This project uses Vue 3 via CDN for simplicity
- No build step or dependencies needed
- Perfect for learning Vue.js basics
